# AI Math School V9

這是適合 GitHub Desktop 的完整版本。

## 最簡單流程

1. 安裝並登入 GitHub Desktop。
2. Clone `CFM0918/math` 到 `D:\GitHub\math`。
3. 雙擊 `安裝V9到GitHub專案.bat`。
4. 選擇 `D:\GitHub\math`。
5. 回 GitHub Desktop 按 `Commit to main`。
6. 再按 `Push origin`。
7. 等待 1～3 分鐘開啟網站。

詳細圖文步驟請雙擊 `發布步驟.html`。

## 注意

安裝器會清除所選 math 專案內的舊網站檔案，但會保留 `.git`，
因此只可選 GitHub Desktop Clone 下來的 `math` 資料夾。
