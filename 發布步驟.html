AI Math School V9

GitHub Desktop 圖形化發布版，不需要 Git Bash、不需要輸入指令。

1安裝 GitHub Desktop

打開瀏覽器搜尋 GitHub Desktop，安裝後登入你的 GitHub 帳號 CFM0918。

2下載 math 專案

在 GitHub Desktop 上方選：

File → Clone repository

選擇 CFM0918/math，Local path 建議：

D:\GitHub\math

按 Clone。

3執行 V9 安裝器

在本 V9 資料夾內，雙擊：

安裝V9到GitHub專案.bat

跳出選擇視窗時，選剛剛下載的：

D:\GitHub\math

安裝器會清除 math 裡目前混亂的舊檔，再完整放入 V9；它會保留 .git，不會刪除 GitHub 連線。

4發布到 GitHub

回到 GitHub Desktop，左下方 Summary 可填：

安裝 AI Math School V9

依序按：

Commit to main → Push origin

5查看網站

等待約 1～3 分鐘，開啟：

https://cfm0918.github.io/math/

若仍看到舊畫面，按 Ctrl + F5。

以後更新教材

把教材放進本機的正確資料夾，例如：

D:\GitHub\math\grades\grade7\semester1\courses\1-1\interactive.html

然後開 GitHub Desktop：

Commit to main → Push origin

不再使用 GitHub 網頁拖放，因此不會再出現 index (1).html 或 interactive (11).html。
