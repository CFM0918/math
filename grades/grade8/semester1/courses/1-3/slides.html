國二上學期・第一章 乘法公式與多項式

1-3　多項式的乘除運算

乘法用分配律逐項展開
除法商出結果後別忘了驗算
被除式 = 除式 × 商式 ＋ 餘式
目標

本節學習目標

  • 用分配律做多項式乘法（逐項相乘再合併）
  • 理解除法定理:被除式 = 除式×商式 + 餘式
  • 用長除法求商式和餘式
  • 會驗算除法結果
一句話:乘法是「不漏項」，除法是「試商→乘→減→帶下一項」，做完一定驗算。
核心
核心　多項式乘法

逐項相乘、不漏項

(a+b)(c+d+e) = ac+ad+ae+bc+bd+be
⚠️ 常見錯:漏項。(a+b)(c+d) 只乘 ac+bd，漏了 ad+bc。
檢查法:m 項 × n 項應該有 m×n 組乘積。
每項都畫線連到另一式的每項，確保沒漏。
例題
例題 1　難度 ★☆☆

展開 (2x−1)(3x²+x−4)

(2x−1)(3x²+x−4)
2x 乘每項　6x³ + 2x² − 8x
−1 乘每項　−3x² − x + 4
合併　6x³ + (2−3)x² + (−8−1)x + 4
6x³ − x² − 9x + 4
例題
例題 2　難度 ★★☆　三個相乘

先算兩個，再乘第三個

(x+2)(x−1)(x+3)
前兩個　(x+2)(x−1) = x² + x − 2
再乘　(x²+x−2)(x+3) = x³+3x²+x²+3x−2x−6
x³ + 4x² + x − 6
三個以上相乘:兩兩逐步做，別想一次乘完。
核心
核心　除法定理

被除式 = 除式 × 商式 ＋ 餘式

被除式 = 除式 × 商式 + 餘式
長除法四步（重複）:從最高次項開始
① 試商（被除式首項 ÷ 除式首項）② 乘 ③ 減 ④ 帶下一項
餘式的次數一定低於除式;餘式為 0 → 整除。
例題
例題 3　難度 ★★★　長除法

(2x³−5x²+3x−1) ÷ (x−2)

步驟動作
①2x³÷x=2x²;2x²(x−2)=2x³−4x²;減 → −x²
②−x²÷x=−x;−x(x−2)=−x²+2x;減 → x
③x÷x=1;1(x−2)=x−2;減 → 1
商式 = 2x²−x+1，餘式 = 1
驗算:(x−2)(2x²−x+1)+1 = 2x³−5x²+3x−1 ✓
例題
例題 4　難度 ★★★　整除＋待定係數

被 (x−1) 整除

(x³+ax²−x+b) 被 (x−1) 整除，商式常數項為 2，求 a、b。
整除　x=1 代入 = 0：1+a−1+b = 0 → a+b = 0
設商　(x²+cx+2)：(x−1)(x²+cx+2) = x³+(c−1)x²+(2−c)x−2
比較　x 項 2−c=−1 → c=3;x² 項 a=c−1=2;常數 b=−2
a = 2、b = −2
核心
🔑 更快的工具

餘式定理 & 因式定理

定理內容
餘式定理P(x) 除以 (x−a) 的餘數 = P(a)
因式定理P(a) = 0 ⟺ (x−a) 是 P(x) 的因式
💡 求餘數不必長除法 —— 直接代入 x=a 算 P(a)!
例:(x³+2x²−x+k)÷(x+3) 餘 5 → 代 x=−3：−27+18+3+k=5 → k=11。
核心
好用的公式

立方和 / 立方差

(a−b)(a²+ab+b²) = a³ − b³
(a+b)(a²−ab+b²) = a³ + b³
(x³−1) ÷ (x−1) = ?
x³−1 = (x−1)(x²+x+1) → 商 = x²+x+1（餘 0）
認出立方和差，除法一秒完成，不必長除。
精選
隨堂精選

快速檢查

題目答案
(2x−1)(x+4)2x²+7x−4
(x²+3x−2)÷(x+1)商 x+2、餘 −4
(x+1)³x³+3x²+3x+1
(x+y+z)²x²+y²+z²+2xy+2yz+2xz
(x⁴−1)÷(x²−1)x²+1
(x⁴−1)=(x²−1)(x²+1)，先分解比長除快。
素養
素養題型 1　【體積】

多項式相乘

長 (2x+3)、寬 (x−1)、高 (x+2) 公分
(1) 體積　(2x+3)(x−1)=2x²+x−3，再×(x+2) = 2x³+5x²−x−6
(2) x=5　250+125−5−6 = 364 立方公分
(3) V÷(x+2)　= 2x²+x−3 = (2x+3)(x−1) = 底面積
素養
素養題型 2　【數論證明】

n³ − n 必被 6 整除

(1) 分解　n³−n = n(n²−1) = (n−1)n(n+1)
(2) 觀察　(n−1)、n、(n+1) 是三個連續整數
(3) 為何被 6 整除　連續三整數必含一個 2 的倍數、一個 3 的倍數 → 積被 2×3=6 整除
💡 因式分解把「數論性質」變成「看得見的結構」。
避雷

避雷指南

❌ 錯誤✓ 正確
乘法漏項m 項×n 項 = m×n 組，數一數
長除法減錯符號減多項式時整體變號
除完不驗算除式×商式＋餘式 = 被除式
用長除法求餘數（慢）餘數定理:餘數 = P(a)
除法做完先問:餘式次數比除式低了嗎?可以驗算了嗎?
總結

本節總結

  • 乘法:逐項相乘不漏項（m×n 組），再合併
  • 除法定理:被除式 = 除式×商式 + 餘式（餘式次數 < 除式）
  • 長除法:試商→乘→減→帶下一項;做完驗算
  • 餘式定理:餘數 = P(a);因式定理:P(a)=0 ⟺ (x−a) 是因式
