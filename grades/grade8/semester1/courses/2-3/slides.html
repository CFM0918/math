國二上學期・第二章 平方根與畢氏定理

2-3　畢氏定理

直角三角形三邊的關係
a² + b² = c²
兩千五百年前的發現，至今無所不在
目標

本節學習目標

  • 記住畢氏定理:兩股平方和 = 斜邊平方
  • 求直角三角形的第三邊（含根號）
  • 用逆定理判斷是不是直角三角形
  • 用畢氏定理算坐標平面兩點距離
一句話:a²+b²=c²，斜邊永遠是最長、在直角對面。解出平方後記得開根號。
核心
核心　畢氏定理

a² + b² = c²

兩股（夾直角的兩邊）平方和 = 斜邊（直角對邊、最長）平方
9 + 16 = 25，這就是 3-4-5。
核心
核心　斜邊 & 逆定理

認清角色再列式

名稱位置
斜邊 c直角的對邊、最長的邊
兩股 a、b夾住直角的兩邊
逆定理:若 a²+b²=c²（c 最長）→ 就是直角三角形。
常見畢氏數組:(3,4,5)、(5,12,13)、(8,15,17)、(7,24,25) 及其倍數。
例題
例題 1、2　難度 ★☆☆~★★☆

求第三邊

兩股 6、8，求斜邊
c² = 6²+8² = 100 → c = √100 = 10（=3-4-5 的兩倍）
斜邊 13、一股 5，求另一股
5²+b² = 13² → 25+b² = 169 → b² = 144 → b = 12
⚠️ 求股時是「斜邊² − 股²」，別加成 13²+5²!
例題
例題 3　難度 ★★☆　逆定理

判斷是否直角三角形

(1) 7、24、25　(2) 5、7、9
(1) 7²+24² = 49+576 = 625 = 25² ✓ → 是直角三角形
(2) 5²+7² = 74 ≠ 81 = 9² ✗ → 不是
判斷法:最長邊的平方 = 另兩邊平方和? 相等才是直角。
例題
例題 4　難度 ★★★　兩點距離

坐標平面上的距離

d = √[(x₂−x₁)² + (y₂−y₁)²]
A(1,2)、B(5,5) 的距離
Δx = 5−1 = 4;Δy = 5−2 = 3
d = √(4²+3²) = √25 = 5
⚠️ Δx、Δy 都要先平方再相加，別漏平方。
例題
例題 5　難度 ★★★　梯子問題

經典應用

5 公尺梯子靠牆，梯底距牆 3 公尺，求梯頂高。再往外推 1 公尺呢?
原高　h₁ = √(5²−3²) = √16 = 4
推出後（距牆 4）　h₂ = √(5²−4²) = √9 = 3
降低　4 − 3 = 1 公尺
梯長固定 = 斜邊;牆高、地距是兩股。

素養
素養題型 1　【斜坡管線】

斜邊 = 實際坡長

水平 24 公尺、垂直 7 公尺
(1) 坡長　√(24²+7²) = √625 = 25 公尺
(2) 費用　25×350 = 8750 元
(3) 坡度　7/24 ≈ 0.292 < 1/3 → 不需加固
(24,7,25) 也是一組畢氏數!

素養題型 2　【廠房對角線】

長方形對角線

長 40 公尺、寬 30 公尺
(1) 對角線　√(40²+30²) = √2500 = 50 公尺
(2) 中央到角落　50 ÷ 2 = 25 公尺
(3) 費用　50×80 = 4000 元
(40,30,50) = (4,3,5)×10。

總結

本節總結

  • a² + b² = c²:兩股平方和 = 斜邊平方（斜邊最長、直角對面）
  • 逆定理:a²+b²=c² → 直角三角形
  • 兩點距離:d = √[(Δx)²+(Δy)²]
  • 畢氏數組:(3,4,5)、(5,12,13)、(8,15,17)、(7,24,25) 背起來省時
🔗 承先:2-1、2-2 根式（斜邊常要開根號）
🔗 啟後:九上三角比、圓的切線、面積計算 —— 畢氏定理是基石。
