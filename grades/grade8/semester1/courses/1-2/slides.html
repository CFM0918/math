1-2　多項式與其加減運算

認識多項式的語言，才能正確整理
加減，就是「合併同類」
5x³ − 3x² + x − 7　（降冪排列的標準式）
目標

本節學習目標

  • 理解多項式:項、係數、次數、常數項、最高次項
  • 判斷兩項是否為同類項
  • 合併同類項 → 化成標準式（降冪）
  • 多項式加減（去括號注意負號分配）
一句話:多項式是代數的骨架;加減的本質就是「把同類項合併」。
核心
核心　多項式的語言

以 5x³ − 3x² + x − 7 為例

名詞說明
項5x³、−3x²、x、−7（四項＝四項式）
係數5、−3、1（x 的係數）、−7
次數最高次項 5x³ → 三次式
常數項−7（次數 0）
標準式按次數由高到低排列
x 前沒寫數字時，係數是 1（不是 0）。
核心
核心　同類項

字母與次方都相同

兩項是否同類
3x²y 與 −5x²y✅ 是
3x²y 與 3xy²❌ 不是（次方不同）
5x² 與 5x❌ 不是（2 ≠ 1）
判斷三問:字母一樣嗎?每個字母的次方也一樣嗎?一個不同就不是同類項。
避雷
核心　兩個關鍵動作

去括號 + 合併

去括號:−(a − b + c) = −a + b − c
⚠️ 全章最高頻錯:括號前是負號 → 括號內「每一項」都要變號，別只翻第一項!
合併同類項:係數相加、變數部分保留。3x²+2x² = 5x²（不是 5!）
例題
例題 1　難度 ★☆☆

辨識多項式

5x³ − 3x² + x − 7
各項　5x³、−3x²、x、−7（四項式）
係數　5、−3、1、−7
次數　最高 5x³ → 三次式　常數項　−7
例題
例題 2　難度 ★★☆　合併同類項

分組再相加

3x²y − 5xy² + 2x²y + 4xy² − xy
x²y 項　3 + 2 = 5 → 5x²y
xy² 項　−5 + 4 = −1 → −xy²
xy 項　−xy
5x²y − xy² − xy
⚠️ x²y 和 xy² 不是同類項，不能合併!
例題
例題 3　難度 ★★☆　減法去括號

負號分配給每一項

(3x² − 2x + 5) − (x² + 4x − 3)
去括號　3x² − 2x + 5 − x² − 4x + 3（每項變號）
合併　(3−1)x² + (−2−4)x + (5+3)
2x² − 6x + 8
−(x²+4x−3) → −x²−4x+3，最後那項 −(−3)=+3 最常錯。
例題
例題 4　難度 ★★★　帶係數

求 2A − 3B

A = 2x²−3x+1，B = x²+x−4，求 2A − 3B。
代入　2(2x²−3x+1) − 3(x²+x−4)
展開　4x²−6x+2 − 3x²−3x+12
x² − 9x + 14
⚠️ −3 要乘進 B 的每一項:−3×(−4) = +12。
例題
例題 5　難度 ★★★　待定係數

比較同次項的係數

(3x²+ax−2) + (bx²−5x+4) = 5x²−3x+2，求 a、b。
x² 項　3 + b = 5 → b = 2
x 項　a + (−5) = −3 → a = 2
常數驗算　−2 + 4 = 2 ✓
a = 2、b = 2
💡 兩多項式相等 → 同次項的係數對應相等。

素養
素養題型 1　【產線成本】

成本多項式加減

C_A = 2x²+50x+8000，C_B = 3x²+30x+6000
(1) 合計　C_A + C_B = 5x² + 80x + 14000
(2) 差值　C_A − C_B = −x² + 20x + 2000
(3) x=100　C_A = 33000、C_B = 39000 → A 線較低
素養
素養題型 2　【加權成績】

代數式化簡

期中 40%、期末 40%、平時 20%;A 為 x、y、z
(1) A 總成績　0.4x + 0.4y + 0.2z
(2) B（期中+5、期末−10、平時+8）　0.4(x+5)+0.4(y−10)+0.2(z+8) = 0.4x+0.4y+0.2z − 0.4
(3) A − B　= 0.4 → A 比 B 多 0.4 分
避雷

避雷指南

❌ 錯誤✓ 正確
減法只翻第一項符號括號前負號 → 每項變號
x²y 與 xy² 當同類項次方也要相同
3x²+2x² = 5= 5x²（保留變數）
沒有降冪排列標準式次數由高到低
合併前先問:這兩項是同類項嗎?
總結

本節總結

  • 多項式語言:項、係數、次數、常數項、標準式（降冪）
  • 同類項:字母與每個字母的次方都相同
  • 減法去括號:括號內每項變號（最高頻錯）
  • 合併:係數相加、變數保留;待定係數:同次項係數對應相等
