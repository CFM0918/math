4-1　因式分解解一元二次方程式

先因式分解，再用「積為零 → 至少一個因式為零」求解
ax² + bx + c = 0　（a ≠ 0）
目標

本節學習目標

  • 理解標準式:ax²+bx+c = 0（a≠0）
  • 理解零積定理:AB=0 → A=0 或 B=0
  • 移項成標準式後，用因式分解求解
  • 會驗算，知道最多兩個解（含重根）
一句話:3-2 練熟的十字交乘，這章用來解方程式 —— 分解＋零積定理就是全部。
核心
核心　零積定理

解方程式的鑰匙

若 A × B = 0，則 A = 0 或 B = 0
⚠️ 只有積等於 0 才能用!
x(x−3)=12 不能寫成 x=12 或 x−3=12（積不是 0）。
兩個數相乘 = 0 → 其中至少一個是 0，這是唯一能「拆開」的情況。
核心
核心　解法四步

移項 → 分解 → 各自=0 → 解

步驟做法
①移項整理成標準式（右邊 = 0）
②左邊因式分解
③每個因式各自等於 0
④解出每個 x
⚠️ 沒移項（右邊≠0）就不能用零積定理!先讓右邊 = 0。
例題
例題 1、2　難度 ★☆☆~★★☆

基本解法

(x−3)(x+5) = 0
x−3=0 或 x+5=0 → x=3 或 x=−5
x²−5x+6 = 0
分解 (x−2)(x−3)=0 → x=2 或 x=3（驗 4−10+6=0 ✓）
⚠️ 解是「x=2 或 x=3」，不是「且」!
避雷
例題 3　難度 ★★☆　最經典的陷阱

x² = 5x 千萬別除 x!

解 x² = 5x
❌ 錯:兩邊除 x → x = 5（漏掉 x=0!）
✓ 對:移項 x²−5x=0 → x(x−5)=0 → x=0 或 x=5
⚠️ 除以含未知數的式子會漏解 —— 永遠先移項再因式分解!
例題
例題 4　難度 ★★★　首項係數≠1

十字交乘登場

2x² + x − 6 = 0
十字交乘　(2x−3)(x+2)=0（交叉 2·2+(−3)·1=1 ✓）
各自=0　2x−3=0 → x=3/2;x+2=0 → x=−2
3-2 的十字交乘，這裡直接派上用場。
例題
例題 5　難度 ★★★　重根

兩個解相等

x² − 6x + 9 = 0
分解　(x−3)² = 0
解　x−3=0 → x=3
💡 這叫「重根」:兩個解都是 3，只有一個相異解。
完全平方式 → 重根。
精選
隨堂精選

快速檢查

方程式解
x²−9 = 0x=±3
x²−7x+12 = 0x=3 或 4
x²+8x+16 = 0x=−4（重根）
3x² = 12xx=0 或 4
(2x−1)² = 9x=−1 或 2
3x²=12x:提 3x，別除!(2x−1)²=9 移項用平方差。
素養
素養題型 1　【長方形面積】

取捨不合理的解

長比寬多 3、面積 40，設寬 x
(1) 列式　x(x+3)=40 → x²+3x−40=0
(2) 解　(x+8)(x−5)=0 → x=−8 或 x=5
(3) 取捨　寬不能為負 → x=5（寬 5、長 8）
⚠️ 應用題要檢查解是否符合實際（長度非負）!
素養
素養題型 2　【利潤函數】

損益兩平與最大利潤

P(x) = −x²+10x−16（萬元），x = 月產量（百條）
(1) 損益兩平 P=0　x²−10x+16=0 → (x−2)(x−8)=0 → x=2 或 8
(2) 獲利範圍　2 < x < 8（200~800 條）
(3) 最大利潤　P=−(x−5)²+9 → x=5 時最大 9 萬元
避雷

避雷指南

❌ 錯誤✓ 正確
x²=5x 兩邊除 x移項 x(x−5)=0（別漏 x=0）
沒移項就用零積定理右邊要 = 0 才能用
「x=3 且 x=−5」是「或」
x(x−3)=12 → x=12積≠0 不能拆，要先移項
追問:右邊等於 0 了嗎?可以用零積定理了嗎?
總結

本節總結

  • 零積定理:AB=0 → A=0 或 B=0（只有積為 0 才能用）
  • 四步:移項（右邊=0）→ 因式分解 → 各自=0 → 解
  • x²=5x 不可除 x:移項提公因式，別漏 x=0
  • 重根:完全平方式 → 兩解相等
