國二上學期・第三章 因式分解

3-1　因式分解提公因式與乘法公式

展開的逆操作
把多項式「收縮」回幾個因式的乘積
展開:(x+2)(x+3) → x²+5x+6
分解:x²+5x+6 → (x+2)(x+3)
目標

本節學習目標

  • 理解因式分解:把多項式寫成因式的乘積（展開的逆）
  • 提公因式法分解
  • 逆用乘法公式:完全平方、平方差
  • 結合多步驟分解（分解到不能再分）
一句話:1-1 學展開，這章學還原。策略永遠是「先提公因式，再套公式，分到底」。
核心
核心　提公因式

找最大公因數提出

6x² + 9x = 3x(2x+3)
怎麼提?
係數　6、9 的公因數 3
變數　x²、x 的公因式 x → 公因式 3x
括號內　6x²÷3x=2x;9x÷3x=3 → (2x+3)
⚠️ 12x²+8x 提 4x → 4x(3x+2)（8÷4=2，不是 8!）
核心
核心　逆用乘法公式

認出公式的「形狀」

因式分解判斷法
a²−b² = (a+b)(a−b)兩項、一正一負、都是平方
a²+2ab+b² = (a+b)²首尾平方、中間兩倍首尾積
a²−2ab+b² = (a−b)²同上，中間負
就是把 1-1 的四個公式反過來讀。
例題
例題 1　難度 ★☆☆　提公因式

提出公因式

(1) 12x³−8x²+4x　(2) 6a²b−9ab²
(1) 公因式 4x → 4x(3x²−2x+1)
(2) 公因式 3ab → 3ab(2a−3b)
檢查:把括號展開回去，應等於原式。
例題
例題 2　難度 ★★☆　平方差

a² − b² = (a+b)(a−b)

(1) x²−25　(2) 4a²−9b²
(1) x²−5² = (x+5)(x−5)
(2) (2a)²−(3b)² = (2a+3b)(2a−3b)
看到「□² − ○²」就想平方差:(和)(差)。
例題
例題 3　難度 ★★☆　完全平方

三項式的判斷

(1) x²+6x+9　(2) 4x²−12x+9
(1) 首 x²、尾 3²、中間 6x=2·x·3 → (x+3)²
(2) 首 (2x)²、尾 3²、中間 12x=2·2x·3 → (2x−3)²
⚠️ 中間項要「剛好」是兩倍首尾積，x²+3x+4 就不是完全平方式!
例題
例題 4、5　難度 ★★★　多步驟

分解到不能再分

2x²−8　·　3x³−12x
2x²−8　先提 2 → 2(x²−4) → 再平方差 → 2(x+2)(x−2)
3x³−12x　先提 3x → 3x(x²−4) → 3x(x+2)(x−2)
(x+y)²−4z²（整體代換）
把 (x+y) 當整體，4z²=(2z)² → (x+y+2z)(x+y−2z)
⚠️ 2(x²−4) 還沒完!每步都問「括號還能分嗎?」

素養
素養題型 1　【輪胎截面積】

環形面積 = π(R+r)(R−r)

截面積 = πR² − πr²
(1) 化簡　π(R²−r²) = π(R+r)(R−r)
(2) R=15,r=12　π×27×3 = 81π ≈ 254.34 cm²
(3) 截面積 63π、R+r=15　15(R−r)=63 → R−r=4.2 → R=9.6、r=5.4

素養題型 2　【平方差速算】

因式分解讓大數變簡單

101² − 99²
(1) 分解　(101+99)(101−99) = 200×2
(2)　= 400
(3) 1001²−999²　= 2000×2 = 4000
💡 避開巨大乘方，只做加減與一次乘法。

總結

本節總結

  • 因式分解 = 展開的逆操作（多項式 → 因式乘積）
  • 提公因式:找最大公因數提出，括號內要除對
  • 逆用公式:平方差 a²−b²、完全平方 a²±2ab+b²
  • 策略:先提公因式 → 再套公式 → 分解到不能再分

3-2　十字交乘法

分解 x² + bx + c 型
找兩數，積為 c、和為 b
x²+5x+6 = (x+2)(x+3)，因為 2×3=6、2+3=5。
交叉相乘，湊出中間項 —— 因式分解最實用的工具。
