4-2　配方法與公式解

因式分解找不到答案時
配方法和公式解都能用 —— 公式解是最後的保險
x = [ −b ± √(b²−4ac) ] / 2a
目標

本節學習目標

  • 用配方法化成 (x+p)² = q
  • 記住公式解:x = [−b±√(b²−4ac)]/2a
  • 用判別式 D=b²−4ac 判斷解的個數
  • 靈活選擇解法
一句話:因式分解優先，配方法次之，公式解萬能 —— 任何一元二次方程式都能解。
核心
核心　配方法

湊成完全平方

x²+6x = (x+3)² − 9（加「一次項係數一半的平方」再減回）
步做法
①移常數項到右邊
②首項係數化 1
③④完成配方 → 開根號 → 求解
⚠️ 首項係數沒化 1 就配方 → 配出錯的完全平方式!
例題
例題 1、4　難度 ★★☆~★★★　配方法

配方法實作

x²−4x−3 = 0
x²−4x=3 → +4 → (x−2)²=7 → x=2±√7
2x²+8x−1 = 0（首項係數化 1）
x²+4x=1/2 → +4 → (x+2)²=9/2 → x=−2±3√2/2
2x² 要先兩邊除 2，變成 x² 再配方。
核心
核心　公式解

最後的保險

x = [ −b ± √(b²−4ac) ] / 2a
⚠️ 代入 a、b、c 符號別出錯（尤其 b 為負時）!
先寫下 a=__、b=__、c=__ 再代入。
任何一元二次方程式（分不分解得開）都能用公式解。
例題
例題 2　難度 ★★☆　公式解

套公式

2x²−5x+1 = 0
辨識　a=2、b=−5、c=1
判別式　D=(−5)²−4·2·1 = 25−8 = 17 > 0（兩相異實數解）
代入　x = [5 ± √17] / 4
⚠️ b=−5 → −b=+5;(−5)²=25（不是 −25）。
核心
核心　判別式 D = b²−4ac

不用解就知道有幾個解

D解的個數
D > 0兩個相異實數解
D = 0重根（一個解）
D < 0無實數解
💡 判別式就在公式解的根號裡 —— 根號內是負的（D<0）就開不出實數。
例題
例題 3、5　難度 ★★☆~★★★　判別式

判斷與求係數

判斷解的個數
x²−4x+4=0:D=16−16=0 → 重根
x²+x+2=0:D=1−8=−7<0 → 無實數解
x²−6x+k=0 有兩相等實根，求 k
重根 → D=0:36−4k=0 → k=9;根 x=3
核心
選擇解法

用哪一招?

情況首選
能整數因式分解因式分解（最快）
不能分解、係數簡單配方法
一般情況公式解（萬能）
先試因式分解;分不開再配方或公式。別每題都硬套公式。
核心
🔑 補充　韋達定理

不解方程式，直接求兩根的和與積

ax²+bx+c=0 的兩根 α、β:
α+β = −b/a　·　α×β = c/a
x²−5x+6=0 的兩根和與積?
α+β = 5、α×β = 6（驗:根為 2、3，2+3=5、2×3=6 ✓）
求「和與積」時，韋達定理比解方程式快得多。
精選
隨堂精選

快速檢查

方程式解 / 判別
x²−3x−4=0（公式）x=4 或 −1
x²−2x+5=0D=−16，無實數解
x²+bx+9=0 重根b=±6
3x²+7x−6=0（公式）x=2/3 或 −3
(x+1)²=3(x+1)+4x=3 或 −2（設 u=x+1）
看到重複的 (x+1) → 設 u 代換，變單純的二次式。
素養
素養題型 1　【自由落體】

h = ½gt²

g≈10，從 100 公尺落下
(1) 列式　5t²=100 → t²=20
(2) 落地時間　t=√20=2√5≈4.47 秒（取正值）
(3) 從 200 公尺　t=2√10，是原來的 √2≈1.41 倍
素養
素養題型 2　【複利】

本利和 P(1+r)²

本金 10 萬，兩年後希望達 11.025 萬
(1) 列式　10(1+r)²=11.025 → (1+r)²=1.1025
(2) 解　1+r=1.05（取正）→ r=0.05
(3) 換算　年利率 5%
總結

本節總結

  • 配方法:移常數 → 首項化 1 → 配方 → 開根號
  • 公式解:x=[−b±√(b²−4ac)]/2a（先寫 a,b,c 再代）
  • 判別式 D=b²−4ac:>0 兩解、=0 重根、<0 無實數解
  • 選擇:能分解→分解;否則配方或公式。韋達:α+β=−b/a、αβ=c/a
