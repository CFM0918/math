3-2　十字交乘法

找兩個數:乘積 = c、和 = b
十字圖幫你快速試出答案
x²+5x+6 = (x+2)(x+3)，因為 2×3=6、2+3=5
目標

本節學習目標

  • 理解原理:(x+a)(x+b) = x²+(a+b)x+ab 的逆向
  • 分解 x²+bx+c 型
  • 處理 ax²+bx+c（首項係數 ≠ 1）
  • 結合提公因式做完整分解
一句話:十字交乘是下一章「解一元二次方程式」的核心工具，這節一定要練熟。
核心
核心　x²+bx+c 型

找兩數:積 c、和 b

x²+bx+c = (x+p)(x+q)，其中 p+q=b、p×q=c
x²+7x+12:找 3、4（積 12、和 7）→ (x+3)(x+4)。
核心
核心　符號判斷

先看常數項 c 的正負

c 符號p、q 符號
c > 0同號（看 b:b>0 都正、b<0 都負）
c < 0一正一負（絕對值大的取 b 的符號）
⚠️ 積、和兩個條件同時滿足;積為負一定一正一負，別把符號搞反。
例題
例題 1、2　難度 ★☆☆~★★☆

x²+bx+c 型

x²+7x+12
p+q=7、pq=12 → (3,4) → (x+3)(x+4)
x²−3x−18
積負 → 一正一負;p+q=−3、pq=−18 → (−6,3) → (x−6)(x+3)
先問:積是幾?和是幾?哪對整數同時滿足?
例題
例題 3　難度 ★★★　首項係數≠1

ax²+bx+c 型

找 (px+r)(qx+s):pq=a、rs=c、ps+qr=b
2x²+7x+3
pq=2、rs=3;試 (2x+1)(x+3):交叉 2×3+1×1=7 ✓
例題
例題 4　難度 ★★★　提+交乘

先提公因式再十字交乘

6x³+9x²−6x
提公因式　3x(2x²+3x−2)
十字交乘　2x²+3x−2 = (2x−1)(x+2)
3x(2x−1)(x+2)
先提公因式讓係數變小，交乘試驗更輕鬆。
例題
例題 5　難度 ★★★　無法分解

何時「整數範圍無法分解」?

x²+x+1 能分解嗎?
需 p+q=1、pq=1
積=1 的整數對:(1,1) 和=2;(−1,−1) 和=−2 → 都不是 1
整數範圍內無法分解
窮舉積的整數對，若沒有一對的和符合 → 無法分解。

隨堂精選

快速檢查

題目分解
x²−8x+15(x−3)(x−5)
x²−x−12(x−4)(x+3)

含兩字母:把 y 當常數，找 p+q=−5y、pq=6y²。
素養
素養題型 1　【面積模型】

面積 → 長 × 寬

長方形面積 x²+8x+15
(1) 分解　(x+3)(x+5) → 長 (x+5)、寬 (x+3)
(2) x=5　長 10、寬 8、面積 80
(3) 2x²+11x+15　= (2x+5)(x+3)
素養
素養題型 2　【連續整數積】

n³ − n 的結構

(1) (n−1)n = n²−n
(2) (n²−n)(n+1) = n³−n
(3) n³−n=24　(n−1)n(n+1)=24 → n=3（2×3×4=24）
連續整數的積 = 因式分解後的乘積形式。

總結

本節總結

  • x²+bx+c:找兩數 p+q=b、pq=c → (x+p)(x+q)
  • 符號:c>0 同號、c<0 一正一負
  • ax²+bx+c:十字交叉 pq=a、rs=c、ps+qr=b
  • 策略:先提公因式 → 再十字交乘 → 分到底;無整數對則無法分解
