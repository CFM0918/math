國二上學期・第二章 平方根與畢氏定理

2-1　平方根與近似值

√ 是「什麼數平方等於它」
不是所有數的平方根都是整數
√2 ≈ 1.414　√3 ≈ 1.732　√5 ≈ 2.236
目標

本節學習目標

  • 理解平方根:x² = a → x 是 a 的平方根
  • 正確用根號:√a 是 a 的「正」平方根
  • 理解實數 = 有理數 ＋ 無理數
  • 用夾擠法求 √a 的近似值
一句話:√a 只取正值;a 的平方根有兩個（±），別把這兩件事搞混。
核心
核心　平方根 vs 根號

±√a 與 √a 不一樣

說法值
9 的平方根±3（兩個）
√9（算術平方根）3（只取正）
⚠️ 最高頻錯:√9 = ±3（✗）。√9 = 3!
正數有兩個平方根;0 只有 0;負數沒有（實數範圍）平方根。
核心
核心　實數的分類

實數 = 有理數 ＋ 無理數

類別小數展開例
有理數有限 / 無限循環2、0.333…、1/3
無理數無限不循環√2、√3、π
⚠️ 無理數是實數 —— 在數線上有確切位置、有大小，不是「沒有大小的數」。
建議記憶:√2≈1.414、√3≈1.732、√5≈2.236。
例題
例題 1　難度 ★☆☆

求平方根

求 49、0.04、1/9 的平方根
49　7²=49 → 平方根 ±7;√49 = 7
0.04　0.2²=0.04 → ±0.2;√0.04 = 0.2
1/9　(1/3)²=1/9 → ±1/3;√(1/9) = 1/3
「平方根」問幾個 → 答 ±;「√」問一個 → 答正的。
例題
例題 2　難度 ★★☆　有理 / 無理

先化簡再判斷

數化簡類別
√4= 2有理數
√5無限不循環無理數
0.333…= 1/3有理數
√12= 2√3無理數
π無限不循環無理數
判斷前先化簡:√4 別急著說無理，它是 2!
例題
例題 3　難度 ★★☆　夾擠法

估算 √7

整數夾　2²=4 < 7 < 9=3² → 2 < √7 < 3
小數一位　2.6²=6.76 < 7 < 7.29=2.7² → 2.6 < √7 < 2.7
更精確　2.64²=6.9696 < 7 < 7.0225=2.65²
√7 ≈ 2.6（到小數一位）
例題
例題 4　難度 ★★★　含字母

√(x²) = |x|，不是 x

化簡 (1) √(x²)（x 任意）　(2) √(4x²)（x≥0）
(1) √(x²) = |x|　（x<0 時 √(x²)=−x=|x|，不能直接寫 x）
(2) √(4x²) = 2|x|;若 x≥0 則 = 2x
⚠️ 根號結果一定 ≥ 0。x 可能是負的，所以要加絕對值!
例題
例題 5　難度 ★★★　提完全平方

不用計算機估大數/小數

已知 √2 ≈ 1.414，求 √200、√0.02
√200　= √(100×2) = 10√2 ≈ 14.14
√0.02　= √(2/100) = √2 ÷ 10 ≈ 0.1414
💡 把完全平方因數提出根號（100=10²），剩下 √2 用近似值。

素養
素養題型 1　【鋼板邊長】

面積開根號

正方形鋼板面積 50 平方公尺
(1) 邊長　√50 = √(25×2) = 5√2 公尺
(2) 近似　5×1.414 = 7.07 公尺
(3) 面積 200　√200 = 10√2 ≈ 14.14 公尺（邊長變 2 倍）
面積變 4 倍 → 邊長變 2 倍（√4=2）。
素養
素養題型 2　【操場對角線】

正方形對角線 = 邊 × √2

正方形操場邊長 100 公尺，測量值 141.2~141.6
(1) 理論對角線　100√2 公尺
(2) 近似　100×1.414 = 141.4 公尺
(3) 最準/最差　第 4 次 141.4（誤差 0）;第 5 次 141.6（誤差 0.2）

避雷指南

√9 = ±3√9 = 3（9 的平方根才是 ±3）
√(x²) = x√(x²) = |x|

總結

本節總結

  • 平方根 vs √:a 的平方根 ±√a（兩個）;√a 只取正（一個）
  • 實數 = 有理數（可寫分數）＋ 無理數（無限不循環）
  • √(x²) = |x|，不是 x
  • 夾擠法估近似;提完全平方因數化簡（√200=10√2）
