國二下學期・第三章 三角形的基本性質

3-1　內角與外角

三角形內角和 180° 是幾何的起點
外角定理是它的直接推論
n 邊形內角和 = (n−2) × 180°
目標

本節學習目標

  • 三角形內角和 = 180°，計算未知內角
  • 外角定理:外角 = 不相鄰兩內角之和
  • 多邊形內角和 = (n−2)×180°
  • 多邊形外角和恆 360°（與邊數無關）
一句話:內角和隨邊數變 (n−2)×180°；外角和永遠是 360°。
核心
核心　三角形

內角和 & 外角定理

∠A + ∠B + ∠C = 180°
外角定理:三角形任一外角 = 不相鄰兩內角之和。
∠ACD（外角）= ∠A + ∠B
⚠️ 是「不相鄰」兩內角，不是相鄰!外角與相鄰內角互為鄰補角（和 180°）。
核心
核心　多邊形內角和

切成 (n−2) 個三角形

n 邊形內角和 = (n−2) × 180°　（n=5→540°、n=6→720°）
⚠️ 是 (n−2)×180°，不是 n×180°!
核心
核心　外角和恆 360°

與邊數無關

任何凸多邊形，外角和 = 360°
正 n 邊形:
每個外角 = 360° ÷ n;每個內角 = (n−2)×180° ÷ n = 180° − 外角
⚠️ 外角和不隨邊數變!三角形、百邊形的外角和都是 360°。
例題
例題 1、2　難度 ★☆☆~★★☆

三角形角度

∠A=55°、∠B=70°，求 ∠C 和外角
∠C=180−55−70=55°;外角=180−55=125°（=55+70 ✓）
∠A 外角 115°、∠B=50°，求 ∠A、∠C
∠A=65°;外角定理 115=50+∠C → ∠C=65°
外角可用「180−內角」或「不相鄰兩內角和」求。
例題
例題 3、4　難度 ★★☆~★★★

正多邊形與求邊數

正十二邊形
內角和 (12−2)×180=1800°;每內角 150°;每外角 30°
內角和 1440°，是幾邊形?
(n−2)×180=1440 → n=10;正十邊形每內角 144°
求邊數:(內角和÷180)+2 = n。
例題
例題 5　難度 ★★★

已知多角求剩一角

五邊形 ∠A=100°、∠B=80°、∠C=120°、∠D=90°，求 ∠E
內角和　(5−2)×180=540°
∠E　540−100−80−120−90=150°
多邊形已知多角 → 用內角和減去已知角。

素養
素養題型 1　【蜂巢密鋪】

為何六邊形能鋪滿?

正六邊形密鋪
(1) 每內角　(6−2)×180÷6 = 120°
(2) 三個共頂點　120×3 = 360° → 恰好填滿一圈，完美密鋪
(3) 正五邊形　108×3 = 324° < 360° → 有空隙，不能密鋪
密鋪條件:內角能整除 360°。

凸七邊形

六個角:120°,135°,115°,125°,130°,140°
(1) 內角和　(7−2)×180=900°
(2) 第七角　900−765=135°
(3) 規範 ≤150°　最大 140°≤150° → 符合

總結

本節總結

  • 三角形內角和 = 180°;外角 = 不相鄰兩內角之和
  • n 邊形內角和 = (n−2)×180°（切成 n−2 個三角形）
  • 外角和恆 360°（與邊數無關）
  • 正 n 邊形:每外角 360°÷n、每內角 180°−外角
