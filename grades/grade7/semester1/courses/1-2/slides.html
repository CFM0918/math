1-2　整數的加減運算

一句「同號加、異號減」，解決所有整數加減
本節導覽

今天要學會四件事

  • ① 整數的加法及加法運算性質　同號／異號規則 ＋ 交換律、結合律
  • ② 整數的減法及減法運算性質　減法轉加法，以及減法「沒有」什麼
  • ③ 整數的加減混合運算　去括號與連續加減
  • ④ 數線上兩點的距離　d = |a − b|
上一節認識了正負數與數線，這一節開始真正的運算。
核心 ①　加法

加法 = 在數線上移動

加正數 → 往右走
加負數 → 往左走
(−2) + (+5) = +3　　從 −2 出發，往右走 5 格
核心 ①　加法

同號相加：符號照抄

符號不變　，　絕對值相加
(+5)+(+3) = +8
都是正號 → 5+3=8，取正
(−5)+(−3) = −8
都是負號 → 5+3=8，取負
🔑
口訣：同號一國，符號照抄，絕對值相加
例題 1 ★☆☆
換你試試看

同號相加

計算 (+5)+(+3) 與 (−5)+(−3)
想法：同號相加 → 符號不變，絕對值相加
(+5)+(+3)：都是正號，5+3=8 → +8
(−5)+(−3)：都是負號，5+3=8，取負號 → −8
答：+8　與　−8
核心 ①　加法

異號相加：大的決定符號

取絕對值較大者的符號　，　絕對值大減小
(+8)+(−3) = +5
|8| > |−3| → 取正號，8−3=5
(−8)+(+3) = −5
|−8| > |3| → 取負號，8−3=5
🔑
口訣：異號打架，絕對值大的贏（決定符號），然後大減小
例題 2 ★★☆
換你試試看

異號相加

計算 (+8)+(−3) 與 (−8)+(+3)
想法：異號 → 比絕對值，誰大取誰的符號，大減小
(+8)+(−3)：|8|=8 > |−3|=3，取正號，8−3=5 → +5
(−8)+(+3)：|−8|=8 > |3|=3，取負號，8−3=5 → −5
答：+5　與　−5
⚠️
最常錯：直接用前面那個數的符號。一定要先比絕對值！
核心 ①　加法運算性質

交換律

a + b　=　b + a

兩個數相加，交換位置，結果不變。

(+3) + (−7) = −4
(−7) + (+3) = −4　　← 位置對調，答案一樣
核心 ①　加法運算性質

結合律與另外兩個性質

結合律　(a + b) + c = a + (b + c)　—— 先加哪兩個都可以
[(−2)+(+5)] + (−6) = (+3) + (−6) = −3
(−2) + [(+5)+(−6)] = (−2) + (−1) = −3　　← 結果相同
加法單位元
a + 0 = a　例：(−7)+0 = −7
相反數相加為 0
a + (−a) = 0　例：(+6)+(−6) = 0
核心 ①　加法運算性質

用性質「湊整」簡化計算

交換律 ＋ 結合律 的真正用途：把好算的湊在一起
計算 (−15) + (+8) + (−3) + (+10)
硬算：−15+8 = −7；−7−3 = −10；−10+10 = 0（要算三次）
聰明算：[(−15)+(−3)] + [(+8)+(+10)]　← 同號湊一起
= (−18) + (+18)
= 0
💡
看到互為相反數的組合（−18 與 +18），直接得 0 —— 這就是性質的威力。
核心 ②　減法

減法轉加法（本節最重要）

a − b　=　a + (−b)

減去一個數　＝　加上它的相反數

這樣一來，所有減法都能用加法的規則處理 —— 不必另外背一套減法規則。
例題 3 ★★☆
換你試試看

減法轉加法

計算 (+6) − (+9) 與 (+6) − (−9)
(+6)−(+9) = (+6)+(−9)　→ 異號，|−9|>|6|，取負，9−6=3 → −3
(+6)−(−9) = (+6)+(+9)　→ 同號，6+9=15 → +15
答：−3　與　+15
⚠️
最常錯：6 − (−9) 算成 6 − 9。減負數要變號，先轉成加法再做！
重要觀念
核心 ②　減法運算性質

減法「沒有」交換律

減法 不能隨便交換位置！
檢驗算式結果
交換律？5 − 3= 2
3 − 5= −2
2 ≠ −2　→　減法沒有交換律
💡
對比：加法 (+3)+(−7) = (−7)+(+3) 成立；但減法一換位置，答案就變相反數。
核心 ②　減法運算性質

減法也「沒有」結合律

(8 − 3) − 2 = 5 − 2 = 3
8 − (3 − 2) = 8 − 1 = 7
3 ≠ 7　→　減法沒有結合律
但是！只要先把減法轉成加法，就可以自由使用加法的交換律與結合律：
8 − 3 − 2 = 8 + (−3) + (−2)　← 全部變加法
= 8 + [(−3)+(−2)] = 8 + (−5) = 3　← 現在可以隨便湊了
🔑
這就是「減法轉加法」真正的價值 —— 把沒有性質的減法，變成有性質的加法。
核心 ③　混合運算

去括號口訣

括號前是「+」→ 去括號後，括號內每項符號不變
+(3 − 8) = 3 − 8
括號前是「−」→ 去括號後，括號內每一項都要變號
−(3 − 8) = −3 + 8
⚠️
最常錯：只變第一項，把 −(3−8) 寫成 −3−8 —— 後面那項忘了變號！
核心 ③　混合運算

計算四步驟

  • ① 先算括號內
  • ② 減法轉加法（全部變成加法）
  • ③ 用交換律／結合律湊整（同號湊一起、相反數湊掉）
  • ④ 由左往右算
照這四步走，再長的算式都不會亂。
例題 4 ★★★
換你試試看

混合運算

計算 5 − (3 − 8) + (−2 − 1)
Step 1　先算括號內：(3−8) = −5；(−2−1) = −3
Step 2　原式 = 5 − (−5) + (−3)
Step 3　5 − (−5) = 5 + 5 = 10
Step 4　10 + (−3) = 7
答：7
核心 ④　數線上兩點的距離

距離公式

d = |a − b| = |b − a|
距離不分方向，所以取絕對值。a−b 和 b−a 互為相反數，絕對值相同。
核心 ④　數線上兩點的距離

三種情況

情況例子計算
兩點都是正數A=+2，B=+9|2−9| = |−7| = 7
兩點都是負數A=−8，B=−3|−8−(−3)| = |−5| = 5
一正一負A=−8，B=+5|−8−5| = |−13| = 13
💡
小技巧：一正一負時，距離 = 兩數絕對值相加（|−8|+|5| = 13）—— 因為要先走回原點，再走出去。
串起 1-1
核心 ④

絕對值原來是「距離」的特例

一個數 a 到原點的距離：d = |a − 0| = |a|
絕對值 = 兩點距離的特例（其中一點正好是原點）
🔗
1-1 學的絕對值，和這一節的距離公式，其實是同一件事。
例題 5 ★★☆
換你試試看

求兩點距離

數線上 A 點是 −8，B 點是 +5，求 A、B 兩點的距離
d = |a − b| = |−8 − 5| = |−13| = 13
驗算（一正一負 → 絕對值相加）：|−8| + |5| = 8 + 5 = 13 ✓
換個順序算也一樣：|5 − (−8)| = |13| = 13 ✓
答：13
例題 6 ★★★

帳戶餘額

小傑帳戶原有 350 元。週一存入 200 元，週二提款 480 元，週三存入 150 元。週三結束後餘額是多少？
列式（存入為正、提款為負）：350 + (+200) + (−480) + (+150)
350 + 200 = 550
550 + (−480) = 70　（異號，550>480，取正，550−480=70）
70 + 150 = 220
答：220 元

素養 · 海洋
生活應用

潛水員的深度追蹤

潛水員從海平面（0 公尺）下潛 18 公尺，再下潛 12 公尺，上浮 7 公尺查看珊瑚礁，最後再下潛 5 公尺。
(1) 算式（下潛為負、上浮為正）：0 + (−18) + (−12) + (+7) + (−5)
(2) 0−18 = −18；−18−12 = −30；−30+7 = −23；−23−5 = −28
(3) 過程位置：0, −18, −30, −23, −28 → 最小值是 −30
答：(2) 海平面下 28 公尺　(3) 最深達海平面下 30 公尺（第二段下潛後）
🤿
注意 (3)：最終深度 ≠ 最深深度 —— 中途上浮過，所以最深點在過程中。

避雷指南

四個最常見錯誤

❌ 錯誤✓ 正確
異號相加直接用前面數字的符號要比絕對值，大的決定符號
6 − (−9) 算成 6 − 9減負數要變號 → 6 + 9 = 15
−(3 − 8) 去括號只變第一項 → −3 − 8每項都要變號 → −3 + 8
以為減法也有交換律5−3 ≠ 3−5，減法沒有交換律
本節總結

四句話記住全部

① 加法：同號符號照抄、絕對值相加；異號取大者符號、絕對值大減小
② 加法性質：交換律、結合律 → 可以湊整簡化計算
③ 減法：a − b = a + (−b)；減法沒有交換律與結合律，但轉成加法後就能用
④ 兩點距離：d = |a − b|；|a| 就是「a 到原點」的距離特例
