1-1　正數、負數與數線

從 0 出發，往兩個方向認識數的世界
本節導覽

今天要學會四件事

  • ① 正負數的概念　用正負號表達「相反意義的量」
  • ② 數線　把抽象的數畫成看得見的位置
  • ③ 數的比較大小　誰大誰小，數線說了算
  • ④ 相反數與絕對值　方向相反、距離相同
這是國中數學的起點 —— 把小學的「0 與正數」擴展到完整的整數世界。
情境引入

為什麼需要負數？

🌡️ 溫度
冷氣房 18°C，南極可到 −40°C。「零下」就是負數的日常說法。
💰 收支
存錢 +200 元，提款 −200 元，餘額 0 表示沒錢也沒欠錢。
⛰️ 海拔
玉山標高 +3952 公尺，海溝最深約 −4000 公尺，海平面是 0。
⚽ 輸贏分
淨勝 +3 分，落後 −3 分，平手是 0 分。
💡
共同點：都有「相反方向」的兩種意義，光靠正數不夠用了。
核心 ①　正負數的概念

正數、負數、零的定義

正數：比 0 大的數。前面可加「+」，通常省略不寫 —— 5 就是 +5。
負數：比 0 小的數。前面一定要加「−」，如 −5，負號不可省略。
0：正數與負數的分界。0 既不是正數，也不是負數。
🔑
正號可省略，負號絕不可省略 —— 這是書寫上最基本的規矩。
最常考迷思
核心 ①

0 到底算什麼？

0
不是正數 不是負數 是分界點
❌ 學生常誤以為「0 不是負的，所以 0 是正數」
❓
追問：「0 前面可以加正號嗎？」讓學生自己發現 0 沒有方向性。
例題 1 ★☆☆
換你試試看

哪些用到了負數？

下列敘述哪些用到負數概念？請寫出對應的數。
(1) 氣溫下降 3 度　(2) 海拔高度 1200 公尺　(3) 提款機領出 500 元　(4) 體重增加 2 公斤
想法：判斷「方向性的量」—— 是增加／上方，還是減少／下方？
(1) 「下降」是減少 → −3　　(2) 海平面之上 → +1200（正數，非負數）
(3) 「領出」帳戶減少 → −500　　(4) 「增加」是正向 → +2
答：用到負數的是 (1) −3 與 (3) −500
⚠️
(2) 是陷阱題 —— 考學生會不會把所有的量都當成負數。
核心 ②　數線

數線的三要素

① 原點：代表 0 的位置　② 正方向：通常向右　③ 單位長度：每一格的長度
數線上每一點對應唯一一個數，反過來每個數也只對應一點（一一對應）。
核心 ②　數線

左小右大

數線上，越往右數字越大，越往左數字越小
例題 2 ★★☆
換你試試看

數線上的點

A 點在 0 的左方 4 格，B 點在 0 的右方 2 格。
(1) 寫出 A、B 代表的數　(2) 哪個比較大？
想法：左方 → 負數，右方 → 正數；格數就是離原點的距離
(1) A 在左方 4 格 → A = −4；B 在右方 2 格 → B = +2
(2) 數線上越右邊越大，B 在 A 的右邊
答：A = −4，B = +2，B 比較大（+2 > −4）
核心 ③　數的比較大小

第一條：正 > 0 > 負

正數　>　0　>　負數

這是最上位的判斷：只要一正一負，正的一定大。

例：+2 > −100　　別被 100 這個大數字騙了！
核心 ③　數的比較大小

第二、三條：同號怎麼比

兩個正數
+8 > +3
數字大的就大（符合直覺）
兩個負數
−8 < −3
數字大的反而小（違反直覺！）
🔑
畫出數線就一目了然：−8 在 −3 的左邊，所以更小。
例題 3 ★★★
換你試試看

判斷對錯（打迷思）

判斷對錯並說明理由：
(1) 0 是最小的正數　(2) 任何負數都比任何正數小　(3) −10 比 −3 大
(1) 錯。0 不是正數，這句話本身矛盾。
(2) 對。數線上負數全在 0 左邊，正數全在右邊。
(3) 錯。−10 在 −3 的左邊，所以 −10 < −3。
答：(1) 錯　(2) 對　(3) 錯
💡
(3) 是全章最大迷思 —— 數字看起來大，不代表數值大。
隨堂練習

小試身手：排大小

題目答案
將 +4、−7、0、−2、+1 由小到大排列−7 < −2 < 0 < +1 < +4
哪些是負數：+3、−8、0、−0.5、6−8、−0.5
寫出一個比 −5 大、比 −1 小的整數−4、−3、−2（共3種）
甲地 −3°C、乙地 +5°C、丙地 0°C，由冷到熱甲 → 丙 → 乙
核心 ④　相反數

什麼是相反數？

只有正負號不同的兩個數，互為相反數
數線意義：互為相反數的兩點，對稱分布在原點兩側，與原點等距。
核心 ④　相反數

相反數的三個重點

數相反數說明
+6−6只改變正負號
−6+6負的相反數是正的
000 的相反數是 0（唯一的特例）
⚠️ 別把相反數和倒數搞混：−3 的相反數是 +3，倒數是 −1/3。
核心 ④　絕對值

絕對值 = 到原點的距離

|a|

a 到原點的距離，恆 ≥ 0

|−5| = |+5| = 5　　距離不分方向，所以絕對值恆非負。
核心 ④　絕對值

求絕對值三規則

|+7| = 7
正數 → 不變
|−7| = 7
負數 → 去負號
|0| = 0
零 → 還是零
❌ 絕不可能 |−7| = −7　　距離不會是負的！
例題 4 ★★☆
換你試試看

距離原點 5 的點

數線上有一點 E，E 到原點的距離是 5。
(1) E 可能是哪些數？　(2) 若 E 在 0 的右邊，E 是多少？
想法：「到原點的距離」就是絕對值 —— 距離相同、方向可有兩種
(1) 左邊是 −5，右邊是 +5
(2) 右邊 → 取正的那個
答：(1) E = −5 或 +5　(2) E = +5
絕對值進階

|x| = a 有幾個解？

情況解例子
a > 0兩個解|x| = 6 → x = +6 或 −6
a = 0一個解|x| = 0 → x = 0
a < 0無解|x| = −3 → ✗（絕對值不可能為負）
🔑
固定提醒句：「絕對值方程式通常有兩個答案，除非等於 0」 —— 別漏掉負的那個！
易錯
絕對值進階

−|x| 的陷阱

−|−6| = ？
Step 1　先算內層：|−6| = 6
Step 2　再看外面的負號：−(6) = −6
答：−|−6| = −6
口訣：層層剝開，由內而外算。−|x| 表示「先取絕對值，再加負號」，結果恆為非正數。
例題 5 ★★★
換你試試看

混合排序

將下列各數由小到大排列：　−7、|−4|、0、−|2|、5
Step 1　先化簡：|−4| = 4；−|2| = −2
Step 2　變成：−7、4、0、−2、5
Step 3　排序：−7 < −2 < 0 < 4 < 5
答：−7 < −|2| < 0 < |−4| < 5
💡
遇到含符號的數列，一律先化簡成真實數值再排。
全章最大迷思
觀念釐清

「數值大小」≠「絕對值大小」

(1) −8 與 −3，哪個大？　(2) |−8| 與 |−3|，哪個大？
(1) 數線上 −3 在 −8 右邊 → −3 較大（−8 < −3）
(2) |−8| = 8，|−3| = 3 → |−8| 較大（8 > 3）
兩個答案完全相反！
發現：負數的絕對值愈大，數值反而愈小。
問「誰比較大」比數值；問「誰離 0 比較遠」比絕對值。

城市氣溫偏離分析

某日 6 城市與「歷史同期平均氣溫」的差距（°C），正值＝偏高，負值＝偏低：
城市台北台中高雄花蓮屏東南投
溫差+1.2−0.8+2.5−1.5+0.3−2.1
(1) 偏冷最多 → 比數值（最負）→ 南投 −2.1
(2) 最接近歷史同期 → 離 0 最近 → 屏東 +0.3
(3) 偏離程度最大（不分冷熱）→ 比絕對值 → 高雄 2.5
答：(1) 南投　(2) 屏東　(3) 高雄

生活應用

登山隊的高度追蹤

登山隊從海平面（0 公尺）出發：第1天爬升至 +850 公尺紮營，第2天下撤 200 公尺，第3天再爬升 600 公尺。地圖上另有一處「低於海平面 50 公尺」的鹽湖紀念碑。
(1) 第2天結束：850 − 200 = +650 公尺
(2) 第3天結束：650 + 600 = +1250 公尺
(3) 鹽湖紀念碑「低於海平面 50 公尺」→ −50，比第2天的 +650 低
答：(1) +650　(2) +1250　(3) −50 公尺，比較低
🗺️
把海平面當成數線的 0，往上為正、往下為負。
素養 · 延伸

絕對值就是「距離」

🏭 品管判定
|實際值 − 標準值| ≤ 容許誤差 → 合格
螺絲 10mm，誤差 ≤0.05：9.97mm → |−0.03| = 0.03 ✓
📏 兩點距離
數線上 a、b 兩點距離 = |a − b|
學校 −8、公園 +5 → |−8 − 5| = 13 公里
💼
在品管、測量、導航中，絕對值用來衡量「偏離多遠」—— 不管偏高還偏低。

本節總結

四句話記住全部

① 正負數：正數 > 0 > 負數；0 是分界，既非正也非負
② 數線：原點、正方向、單位長度；左小右大
③ 比大小：正 > 0 > 負；兩負數絕對值大的反而小
④ 相反數與絕對值：相反數＝只差正負號、對稱等距；絕對值＝到原點的距離，恆 ≥ 0
