4-1　一元一次不等式

不等式和方程式的差別:
解不是一個數，而是一個「範圍」
x > 2　（2 以右的所有數）
目標

本節學習目標

  • 理解不等式的意義（描述兩個量的大小關係）
  • 能把文字條件列成不等式（至多、至少、不超過…）
  • 能判斷一個數是不是解（代入）
  • 能用數線表示解集合（實心／空心圓、箭頭）
  • 掌握基本性質，尤其 ⚠️ 乘除負數要翻轉不等號
一句話:不等式幾乎和方程式一樣，只有一個致命差別 —— 乘除負數要翻轉。
核心
核心觀念

不等式的解是「範圍」

　方程式不等式
符號=> < ≥ ≤
解一個數（x=4）一個範圍（x>2）
數線一個點一段區間
💡 「x > 2」代表 2.1、3、100…無限多個數都符合。
核心
基礎

四個不等號

符號讀法含端點嗎
>大於❌ 不含
<小於❌ 不含
≥大於等於（不小於）✅ 含
≤小於等於（不大於）✅ 含
關鍵:有沒有「等號」那條線，決定含不含端點
—— 這在數線上就是實心 vs 空心。
核心
細目 ①　列式（新增）

關鍵詞翻譯:看「臨界詞」

中文不等號含端點
超過、多於、大於>❌
不足、少於、小於<❌
至少、不少於、不低於≥✅
至多、不超過、不高於≤✅
⚠️ 「超過 5」→ x > 5（不含）;「至少 5」→ x ≥ 5（含）
有沒有「等於」的味道，決定用 > 還是 ≥。
例題
例題　難度 ★☆☆　列式

把文字翻成不等式

情境不等式
年齡至少 18 歲x ≥ 18
速度不超過 60v ≤ 60
重量超過 50 公斤w > 50
分數不足 60s < 60
誤差不超過 0.2|T − 36.5| ≤ 0.2
📌
列式的重點不是計算，是讀懂中文。「至少」「至多」「超過」「不足」—— 這些詞決定用哪個不等號、含不含端點。圈出臨界詞再列式。
例題
細目 ②　判斷解

代入看成不成立

x=3、x=2 是否為 2x+1 > 5 的解?
x=3　2×3+1 = 7，7 > 5 ✓ → 是解
x=2　2×2+1 = 5，5 > 5 ✗（要「大於」，等於不算）→ 不是解
⚠️ 邊界要小心:x=2 讓左邊剛好 = 5。
因為是嚴格大於（>），等於不算 → 不是解。
若是 ≥，x=2 就會是解。
核心
細目 ②　數線表示

實心／空心、箭頭方向

要素規則
圓點有等號（≥≤）→ 實心 ●;沒等號（><）→ 空心 ○
箭頭大於（>≥）→ 向右;小於（<≤）→ 向左
記法:有等號就填滿（實心），沒等號就留空（空心）
x > 2（空心・向右）
x ≤ −1（實心・向左）
核心
基本性質

運算時，方向會不會變?

運算方向
兩邊同加、同減同一個數不變
兩邊同乘、同除 正數不變
兩邊同乘、同除 負數⚠️ 翻轉！
加減、乘除正數 —— 和方程式一模一樣。
唯一的差別是 乘除負數。
避雷
🔑 本章最重要的規則

乘除負數，方向翻轉

a > b，兩邊同乘 −2 → −2a < −2b（> 變 <）
為什麼?用具體數字看:
3 > 2，但兩邊乘 −2 → −6 < −4（大小顛倒了！）
乘負數會「翻到數線另一邊」→ 大小關係顛倒。
⚠️ 這是整個不等式單元最高頻的錯誤。
固定儀式:每次乘除前先問「正數還是負數?負數 → 翻轉」。
例題
例題 3　難度 ★★☆

性質判斷

已知 a > b，判斷下列是否成立
(1) a+3 > b+3　同加 3，方向不變 → 成立
(2) −2a > −2b　同乘 −2（負數！）→ 應翻轉成 −2a < −2b → 不成立
(3) a/3 > b/3　同除 3（正數），方向不變 → 成立
💡 只有 (2) 動到負數 → 只有 (2) 要翻轉。其他都不變。
例題
範圍的運算

給 x 範圍，求另一個式子的範圍

已知 −3 < x ≤ 5，求 −2x 的範圍
兩邊同乘 −2（負數！）→ 方向全部翻轉
−3×(−2) = 6;5×(−2) = −10
原本 −3 < x ≤ 5 → 翻轉後 −10 ≤ −2x < 6
⚠️ 注意端點也跟著走:原本 −3 那端是 <（開）、5 那端是 ≤（閉）→ 乘 −2 後 6 那端變開、−10 那端變閉。
精選
隨堂精選　★★☆

混合運算:先乘後加

已知 −1 < x ≤ 3，求 2x+1 的範圍
先乘 2（正數，不變）　−2 < 2x ≤ 6
再加 1（不變）　−1 < 2x+1 ≤ 7
−1 < 2x+1 ≤ 7
💡 乘正數、加減都不翻;只有乘除負數才翻。
這題全程沒碰負數，所以方向一路不變。
避雷
隨堂精選　⚠️ 陷阱

「a > b ⟹ a² > b²」→ 錯

判斷:「若 a > b，則 a² > b²。」
反例　a=1、b=−3
a > b:1 > −3 ✓
a² = 1、b² = 9 → 1 < 9 → a² < b² ✗
💡 有負數時，平方的大小不能直接由原數判斷 ——
因為平方把負號吃掉了（−3 平方變成 9，比 1 大）。
📌
用反例最有力:不確定一個不等式規則對不對，就代含負數的數字試。一個反例就推翻它。
素養
素養題型 1　【溫度誤差】

絕對值不等式

測量誤差 |T − 36.5| ≤ 0.2（°C）
(1) 意思　T 與 36.5 的差的絕對值不超過 0.2 → T 在 36.5 附近 ±0.2
(2) 36.8°C?　|36.8 − 36.5| = 0.3 > 0.2 → 不在範圍
(3) 範圍　−0.2 ≤ T − 36.5 ≤ 0.2，各加 36.5 → 36.3 ≤ T ≤ 36.7
💡 |x| ≤ a 就是 −a ≤ x ≤ a（距離不超過 a）—— 絕對值不等式的關鍵。
素養
素養題型 2　【胎壓標準】

誤差範圍

標準 32 PSI、允許誤差 ±1.5 PSI，胎壓 p
(1) 合格　|p − 32| ≤ 1.5，即 30.5 ≤ p ≤ 33.5
(2) p=33.8?　33.8 > 33.5 → 不合格
(3) p=30.2?　|30.2 − 32| = 1.8 > 1.5 → 偏離 1.8 PSI → 不合格
💡 「標準值 ± 誤差」就是 |實測 − 標準| ≤ 誤差，展開就是一段範圍。
避雷

避雷指南:五個最常見錯誤

❌ 錯誤✓ 正確
乘除負數忘記翻轉不等號乘除負數，方向必翻（本章最重要！）
> 用實心、≥ 用空心有等號實心 ●、沒等號空心 ○
把解當成一個數解是一個範圍（數線一段區間）
「超過 5」寫成 x ≥ 5超過 = >（不含）;至少才是 ≥
a > b 就推 a² > b²有負數時不成立（反例 1 > −3）
自我檢查句:「這一步乘除的是正數還是負數?負數要翻轉！」
總結

本節總結

  • 不等式的解是範圍，不是一個數 —— 和方程式最大的差別
  • 列式:看臨界詞 —— 超過/少於用 ><（不含）、至少/至多用 ≥≤（含）
  • 數線:有等號實心 ●、沒等號空心 ○;大於向右、小於向左
  • ⚠️ 乘除負數，方向必翻 —— 整章最高頻的錯，每次乘除前先問「正還是負?」

4-2　不等式的解法與應用

會解方程式，就會解不等式 ——
只要記得乘除負數翻轉
解 −2x + 3 > 11:
移項 → −2x > 8 → 兩邊除以 −2（負數！）→ x < −4（翻轉！）
步驟和解方程式一模一樣，只有除負數那一步要翻。
