國一下學期・第一章 二元一次聯立方程式

1-2　聯立方程式的解

兩條方程式合在一起 —— 唯一解就跑出來了
核心只有一個字:消去
目標

本節學習目標

  • 理解聯立方程式的解:同時滿足兩條的唯一一組 (x, y)
  • 能用代入消去法求解
  • 能用加減消去法求解(含先乘倍數再消)
  • 能判斷哪種方法比較快
  • 知道聯立方程式的三種情形:唯一解／無解／無限多解
一句話:兩個未知數太難 —— 那就先消掉一個,變回七上的題目。
核心
核心思想

解聯立的唯一祕訣:消去

兩個未知數太難處理 → 先消掉一個 → 變成七上的一元一次方程式
步驟動作
① 消去用代入或加減,把一個未知數消掉
② 解一元剩下一個未知數 → 就是七上的一元一次
③ 回代把解出的值代回去,求另一個未知數
④ 驗算代回兩條原方程式
⚠️ 最常漏的是 ③ 回代 —— 消去後解出一個,就以為做完了。
核心
細目 ②　代入消去法

從一條解出 y=…,代入另一條

適用時機:其中一條已經是 y = …,或容易解出(有係數 1 的未知數)
解 { y = 2x，x + y = 12 }
代入　第一式已是 y = 2x → 代入第二式:x + 2x = 12
解　3x = 12 → x = 4
回代　y = 2×4 = 8
驗算　4 + 8 = 12 ✓
x = 4，y = 8
💡 這就是 1-1 的「解出 y = …」派上用場的地方 —— 那時說它是「入場券」,現在兌現了。
例題
例題 2　難度 ★★☆　代入

沒有現成的 y=… 怎麼辦?

解 { 2x + y = 10，x − y = 2 }
Step 1　兩條都沒有現成的 y=…,但第二式 x 係數是 1 → 解出 x = y + 2
Step 2　代入第一式:2(y + 2) + y = 10 → 2y + 4 + y = 10 → 3y = 6 → y = 2
Step 3　回代:x = 2 + 2 = 4
驗算　2×4 + 2 = 10 ✓;4 − 2 = 2 ✓
x = 4，y = 2
💡 選係數是 1 的未知數來解(1-1 學的)—— 這樣代入後不會出現分數。
核心
細目 ③　加減消去法

🔑 口訣:同號相減、異號相加

適用時機:某個未知數的係數相同或互為相反數
兩式中某未知數的係數怎麼消
相同(都是 +2y)相減(+2y − 2y = 0)
互為相反(+3y 與 −3y)相加(+3y + (−3y) = 0)
⚠️ 最常搞反:同號要「減」、異號要「加」
記法:要讓那項變 0,就想「怎樣加減會抵消」。
例題
例題 3、4　難度 ★★☆～★★★　加減

相同 → 相減　·　互為相反 → 相加

例題3　{ 3x + 2y = 16，x + 2y = 8 }
y 係數都是 +2(相同)→ 相減:(3x+2y) − (x+2y) = 16 − 8 → 2x = 8 → x = 4
回代:4 + 2y = 8 → y = 2
x = 4，y = 2
例題4　{ 2x + 3y = 12，4x − 3y = 6 }
y 係數 +3 與 −3(互為相反)→ 相加:6x = 18 → x = 3
回代:6 + 3y = 12 → y = 2
x = 3，y = 2
核心
細目 ③　進階

係數既不同也不相反 → 先乘倍數

隨堂第 8 題:「加減消去法只能用在係數完全相同的情況」→ 錯
{ 3x + 2y = 7，2x + 3y = 8 }　—— y 的係數是 2 和 3,不同也不相反
先湊成一樣　第一式 ×3、第二式 ×2 → y 的係數都變 6
　　9x + 6y = 21
　　4x + 6y = 16
相減　5x = 5 → x = 1;回代 → y = 2
x = 1，y = 2
💡 這叫擴分的逆用 —— 和 2-2 通分「湊成共同分母」是同一個腦子。
所以加減消去法幾乎任何聯立都能用,只是有時要先乘倍數。
工具

怎麼選:代入 vs 加減?

看到什麼用哪個
有一條是 y = …(或 x = …)代入
有係數是 1 的未知數代入(先解出它)
某未知數係數相同或互為相反加減(直接消)
係數都很醜加減(乘倍數湊)
追問句:「這兩條裡,哪個未知數的係數比較適合直接消去?」
動筆前先觀察三秒,選對方法省一半時間。
📌
兩種方法一定得到同一個答案 —— 選錯只是慢,不會錯。但養成「先看」的習慣,考試差很多。
補給
［補給站］解的情形　⚠️ 本節新增

聯立方程式不一定有唯一解

情形一:唯一解(最常見)
{ x + y = 8，x − y = 2 } → (5, 3) 一組
兩條互相獨立、資訊不同 → 鎖定一個點
情形二:無解
{ x + y = 5，x + y = 8 }
同樣的 x+y,不可能既等於 5 又等於 8 → 矛盾 → 無解
情形三:無限多解
{ x + y = 5，2x + 2y = 10 }
第二式 = 第一式 ×2 → 其實是同一條(1-1 說的「不獨立」)→ 無限多解
補給
［補給站］

🔑 怎麼分辨?消去之後看結果

消去後得到情形意思
x = 某數(正常)唯一解鎖定一組
0 = 非零數(如 0 = −3)無解矛盾,不可能
0 = 0(恆成立)無限多解兩條其實是同一條
💡 這正是 1-1「自由度」的延續:
・兩條獨立 → 自由度 0 → 唯一解
・兩條矛盾 → 無解
・兩條不獨立(其實一條)→ 自由度仍是 1 → 無限多解
伏筆
［補給站］🔗 2-2 的伏筆

2-2 會讓你「看見」這三種情形

每條二元一次方程式在坐標平面上是一條直線
代數(這節)幾何(2-2 會講)
唯一解兩直線相交於一點
無解兩直線平行(永不相交)
無限多解兩直線重合(完全疊在一起)
代數的「無解」就是幾何的「平行」;「無限多解」就是「重合」。
例題
例題 5　難度 ★★★

硬幣問題 —— 兌現 1-1 的伏筆

10 元硬幣 x 個、50 元硬幣 y 個,共 8 個、320 元
{ x + y = 8，10x + 50y = 320 }
代入　x = 8 − y → 10(8−y) + 50y = 320
　　80 − 10y + 50y = 320 → 40y = 240 → y = 6
回代　x = 8 − 6 = 2
驗算　2 + 6 = 8 ✓;10×2 + 50×6 = 320 ✓
10 元硬幣 2 個,50 元硬幣 6 個
💡 1-1 結尾叫你「先猜猜看」,答案就是 (2, 6) —— 現在你能算出來了。
避雷
⚠️ 減負變加

相減時最容易錯的一步

{ 3x + 2y = 19，3x − y = 13 }　—— x 係數都是 +3,相減消 x
相減　(3x + 2y) − (3x − y) = 19 − 13
關鍵　2y − (−y) = 2y + y = 3y　← 減去負的要變加!
　　3y = 6 → y = 2
回代　3x + 4 = 19 → x = 5
x = 5，y = 2
⚠️ 2y − (−y) 很多人算成 2y − y = y → 錯
正確 2y + y = 3y(七上 1-2「減負變加」的鐵則)
素養
素養題型 1、2　🔗 完整閉環

兌現 1-1 驗證過的答案

素養1　雞兔共 35 隻、腳共 94 隻(1-1 驗證過 (23,12))
{ x+y=35，2x+4y=94 } → 代入 x=35−y → 70+2y=94 → y=12;x=23
(23, 12) —— 與 1-1 完全一致!
素養2　售票共 150 張、收入 24000(1-1 驗證過 (90,60))
先化簡第二式(÷100):{ x+y=150，2x+y=240 }
相減:x=90;回代 y=60
(90, 60) —— 與 1-1 一致!
💡 這就是 1-1 → 1-2 的完整閉環:
1-1 教你「怎麼確認一組數是不是解」;1-2 教你「怎麼直接算出那組數」。
避雷

避雷指南:五個最常見錯誤

❌ 錯誤✓ 正確
加減時同號用「加」、異號用「減」同號相減、異號相加(想「怎樣抵消」)
消去後解出一個就交卷一定要回代求另一個
只驗算一條方程式代回兩條原方程式
相減時 2y − (−y) 算成 2y − y減去負的要變加:= 2y + y = 3y
以為加減法只能用在係數相同時係數不同可先乘倍數湊
自我檢查句:「這兩條裡,哪個未知數的係數最適合直接消去?」
總結

本節總結

  • 核心是「消去」:把兩個未知數變成一個,退化成七上的一元一次
  • 代入適合有 y = …;加減適合係數相同(相減)或互為相反(相加)
  • 口訣:同號相減、異號相加;係數都醜就先乘倍數湊
  • ［補給站］三情形:消去後 x=某數(唯一)、0=非零(無解)、0=0(無限多解)
🔗 承先:1-1 的「解出 y=…」是代入第一步;1-1 的「自由度」解釋三情形
🔗 啟後:1-3 用這節解應用題;2-2 讓你看見 —— 唯一解=相交、無解=平行、無限多解=重合
