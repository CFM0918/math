2-1　平面上的直角坐標

一對數字 (x, y) 就能定位平面上任何一點
——這是圖形與代數連接的起點
目標

本節學習目標

  • 理解直角坐標平面的構成:x 軸、y 軸、原點、四個象限
  • 能標點(給 (x,y) 畫出來)也能讀點(從圖上讀坐標)
  • 能判斷一個點落在哪個象限(或哪條軸上)
  • 能算平行軸兩點的距離(水平 |x₁−x₂|、垂直 |y₁−y₂|)
一句話:坐標把「幾何的點」和「代數的數對」綁在一起 —— 這是整章的地基。
核心
承接 1-2

為什麼要有坐標?

一對數字 (x, y) 就能定位平面上任何一點
回想 1-2 的結尾:
你解聯立方程式算出 (x, y),我說「它是兩條直線的交點」。
但那時你看不到那個點 —— 因為還沒有坐標平面。
這一章就是要給你那張圖。
核心
細目 ①

直角坐標平面的四個零件

x 軸:水平的數線(往右 +)
y 軸:垂直的數線(往上 +)
原點:兩軸交點 (0, 0)
象限:兩軸把平面分成四塊
坐標 (x, y):x 是橫向(右+左−)、y 是縱向(上+下−)。順序固定:先 x 後 y。
避雷
細目 ①　⚠️ 最常見錯誤

先 x 後 y —— 別寫反了

❌ 把 (3, 5) 標在 x=5、y=3 的位置
✓ 正確 (3, 5)
先往右走 3 格
再往上爬 5 格
記法:「先走廊(橫),再爬樓梯(縱)」 —— 先左右,再上下。
核心
細目 ②　四個象限

從右上開始,逆時針數

象限xy符號
第一象限++(+, +)
第二象限−+(−, +)
第三象限−−(−, −)
第四象限+−(+, −)
順序:右上 → 左上 → 左下 → 右下(逆時針)
核心
細目 ②　⚠️ 重要

軸上的點不屬於任何象限

x = 0 → 在 y 軸上　　y = 0 → 在 x 軸上
(0, 3):x = 0 → 在 y 軸上
不是第一象限!

(−5, 0):y = 0 → 在 x 軸上
這些點是各象限的邊界 —— 就像「牆」不屬於左邊或右邊的房間。
例題
例題 1　難度 ★☆☆

判斷象限

A(3,5)　(+,+) → 第一象限
B(−2,4)　(−,+) → 第二象限
C(−1,−3)　(−,−) → 第三象限
D(4,−2)　(+,−) → 第四象限
E(0,3)　x=0 → 在 y 軸上(不屬於任何象限)
F(−5,0)　y=0 → 在 x 軸上
判斷象限只看兩個坐標的正負號 —— 但先檢查有沒有 0(在軸上)。
例題
例題 3　難度 ★★☆

從象限反推符號

A(a, 3) 在第二象限、B(2, b) 在第四象限,判斷 a、b 的正負
A 在第二象限 → (−, +);y=3 已是正 → a 必須為負 → a < 0
B 在第四象限 → (+, −);x=2 已是正 → b 必須為負 → b < 0
a < 0，b < 0
解法:先寫出那個象限的符號模板(−,+),再對照哪個坐標還沒確定。
📌
進階:「(a,b) 在第二象限,(−a,−b) 在哪?」
a<0→−a>0、b>0→−b<0 → (正,負) → 第四象限。加負號 = 對原點對稱(二↔四、一↔三)。
核心
細目 ①　距離

平行軸兩點的距離:只有兩種能算

水平線段(y 相同)
距離 = |x₁ − x₂|
P(−3,2)、Q(4,2)
y 都是 2 → |−3−4| = 7
垂直線段(x 相同)
距離 = |y₁ − y₂|
A(1,5)、B(1,−3)
x 都是 1 → |5−(−3)| = 8
⚠️ 一定加絕對值(距離不可能是負)。|5−(−3)| = |5+3| = 8(減負變加!)
🔑 追問句:「這兩點是 x 一樣還是 y 一樣?」
y 一樣 → 算 x 的差(水平);x 一樣 → 算 y 的差(垂直)
例題
例題 4　難度 ★★★

直角三角形三邊(畢氏定理預備)

A(1,5)、B(1,−3)、C(4,−3)
AB　x 都是 1(垂直)→ |5−(−3)| = 8
BC　y 都是 −3(水平)→ |1−4| = 3
AC　斜邊 → √(8²+3²) = √73
⚠️ 斜邊要用畢氏定理(八上內容)—— 本節先記算式。
例題
例題 5　難度 ★★★

判斷形狀＋周長面積

A(−2,1)、B(3,1)、C(3,−4)、D(−2,−4) 圍成?
四邊方向　AB 水平、BC 垂直、CD 水平、DA 垂直 → 四個直角
邊長　AB=|−2−3|=5;BC=|1−(−4)|=5 → 都是 5
結論　四邊相等 + 直角 → 正方形!
周長 = 5×4 = 20;面積 = 5×5 = 25
精選
隨堂精選　★★☆

距離已知,反求坐標 → 兩個答案

P(x, 3) 與 Q(2, 3) 的距離是 7,求 x
觀察　y 都是 3 → 水平 → |x − 2| = 7
兩種情況　x − 2 = 7 → x = 9
　　或 x − 2 = −7 → x = −5
x = 9 或 x = −5
💡 絕對值方程式有兩個解 —— Q 的左右各一個點,距離都是 7。
素養
素養題型 1　【城市規劃】

L 形 vs 直線

市政廳為原點,每格 1 公里。圖書館(−3,4)、公園(5,4)、學校(5,−2)。
(1) 圖書館→公園　y 都是 4 → |−3−5| = 8 公里
(2) 公園→學校　x 都是 5 → |4−(−2)| = 6 公里
(3) L 形路線　8 + 6 = 14 公里
　　直線 = √(8²+6²) = √100 = 10 公里 → 多走 4 公里
💡 (3) 藏了一個畢氏定理(8-6-10 直角三角形)—— 八上會學。
現在你先感受:繞路(L 形)一定比直線遠。
伏筆
🔗 兌現 1-2 的伏筆

解 = 交點,現在你能畫出來了

1-2 的硬幣問題解出 (2, 6)
在坐標平面上,就是 x=2、y=6 的那個點。
而 2-2 會讓你看到:x+y=8 和 10x+50y=320 各是一條直線,
它們就相交在 (2, 6)。
代數的「解」= 幾何的「點」
代數的「方程式」= 幾何的「線」
避雷

避雷指南:五個最常見錯誤

❌ 錯誤✓ 正確
把 x、y 寫反((3,5) 標成 x=5,y=3)先 x 後 y(先走廊再爬樓梯)
把 (0,3) 說成第一象限軸上的點不屬於任何象限
算距離忘記絕對值,得到負數距離一定 ≥ 0,要加 |·|
|5−(−3)| 算成 |5−3|=2減負變加:|5+3| = 8
象限順序數錯從右上逆時針:一、二、三、四
自我檢查句:「這兩點是 x 一樣還是 y 一樣?」
總結

本節總結

  • 坐標 (x, y):先 x(橫)後 y(縱);分成四個象限,軸上的點不屬於任何象限
  • 平行軸的距離:y 同算 x 差、x 同算 y 差;一定加絕對值
  • 象限符號:一(+,+)、二(−,+)、三(−,−)、四(+,−);逆時針
  • 加負號 = 對原點對稱(二↔四、一↔三)

🔗 啟後:2-2 讓方程式變成直線 —— 你會看到 1-2 的三種解就是兩直線的相交/平行/重合

2-2　二元一次方程式的圖形

一個方程式的所有解 ——
連起來就是一條直線
2x + y = 10 有無限多組解(1-1 學過)——
把它們全部畫在坐標平面上,會排成一條直線。
而兩個方程式的交點,就是聯立方程式的解。
