國一下學期・第四章 一元一次不等式

4-2　解法與應用

會解方程式，就會解不等式 ——
只要記得一件事:乘除負數，方向翻轉
−2x + 3 > 11 → −2x > 8 → x < −4（翻！）
目標

本節學習目標

  • 能用移項法則解一元一次不等式
  • 能正確處理 ⚠️ 乘除負數時不等號翻轉
  • 能解含括號、含分母的不等式
  • 能把文字（不超過/至少/超過…）翻譯成不等式並列式
  • 能依情境對解做整數限制；能解連立（雙邊）不等式取交集
一句話:解法和解方程式一模一樣，只有除以負係數那一步要翻轉。
核心
核心觀念

解法流程:和方程式只差一步

步驟做法
① 去括號用分配律展開
② 去分母兩邊乘 LCM（正數，方向不變）
③ 移項合併x 項一邊、常數一邊（移項變號）
④ 除以係數係數是負數 → ⚠️ 不等號翻轉
⑤ 數線表示空心/實心 + 箭頭
唯一的差別在 第 ④ 步 —— 其他步驟和解方程式完全一樣。
避雷
🔑 本章的靈魂

乘除負數，方向翻轉

−2x ≤ 6　兩邊 ÷(−2)、翻轉　→　x ≥ −3
為什麼?用具體數字看: 3 > 2，兩邊乘 −2 → −6 < −4（大小顛倒！）
固定儀式:每次乘除前先問「正數還是負數?負數 → 翻轉」。
例題
例題 1　難度 ★☆☆

解 3x + 5 > 14

解不等式：3x + 5 > 14
移項　3x > 14 − 5 = 9
除以 3（正數，方向不變）　x > 3
x > 3
3 處空心圓（> 不含端點），向右箭頭。
例題
例題 2　難度 ★★☆　⚠️ 負係數

解 −2x + 3 ≤ 9

解不等式：−2x + 3 ≤ 9
移項　−2x ≤ 9 − 3 = 6
除以 −2（負數！方向翻轉）　≤ 變 ≥ → x ≥ −3
x ≥ −3
−3 處實心圓（≥ 含端點），向右箭頭。除負數 → 翻！
例題
例題 3　難度 ★★☆　含括號

解 2(x+3) − 5 > x + 4

解不等式：2(x+3) − 5 > x + 4
去括號　2x + 6 − 5 > x + 4 → 2x + 1 > x + 4
移項　2x − x > 4 − 1 → x > 3
x > 3
有括號先展開，再移項 —— 全程沒碰負係數，方向不變。
例題
例題 4　難度 ★★★　含分母

解 x/2 − 1 < (x+3)/3

解不等式：x/2 − 1 < (x+3)/3
去分母　兩邊乘 6（LCM，正數不翻）　3x − 6 < 2(x+3)
展開　3x − 6 < 2x + 6
移項　x < 12
x < 12
⚠️ 去分母時每一項都要乘 LCM（含常數項 −1 → −6）。
例題
例題 5　難度 ★★★　負係數+括號

解 −3(2x−1) ≥ 4x − 5

解不等式：−3(2x−1) ≥ 4x − 5，並用數線表示。
去括號　−6x + 3 ≥ 4x − 5
移項　−6x − 4x ≥ −5 − 3 → −10x ≥ −8
除以 −10（負數，翻轉）　x ≤ 8/10 = 4/5
x ≤ 4/5
4/5 處實心圓，向左箭頭。
精選
隨堂練習　解法快照

自我檢查:方向對了嗎?

題目答案
−4x ≥ 12x ≤ −3（÷−4 翻）
−(x+3) ≤ 5x ≥ −8
x/3 + 1 > 2x > 3
−5x > 10 ⟹ x > −2 ?錯！應 x < −2
−2(x−4) > 3(x+2)x < 2/5
只要式子出現負係數，就在旁邊畫「↺」提醒翻轉。
核心
應用的關鍵

把文字翻成不等號

文字不等號端點
超過、多於、大於>❌ 不含
少於、小於、不足<❌ 不含
至少、不少於、不低於≥✅ 含
至多、不超過、不多於≤✅ 含
記憶法:「至少 = 下限 = ≥」「不超過 = 上限 = ≤」 —— 最常被弄反的兩組。
避雷
應用的陷阱

依情境取整數

人數、件數、次數 → 解出的範圍必須取整數
解出範圍後怎麼取整數?
x > 3.4（至少幾個）→ 取最小整數 x = 4
x ≤ 6.67（最多幾個）→ 取最大整數 x = 6
⚠️ 常見錯誤:解出小數直接當答案（「至少 3.4 個」）—— 沒依情境取整數。
例題
例題 6　難度 ★☆☆　列式應用

最多能買幾個包子?

每天最多花 40 元。包子 5 元、豆漿 10 元，固定買 1 杯豆漿，最多買幾個包子?
設 買 x 個包子　列式　5x + 10 ≤ 40
解　5x ≤ 30 → x ≤ 6
最多 6 個包子
「最多花 40」→ ≤ 40（不超過，含端點）。
例題
例題 7　難度 ★★☆　整數限制

至少要答對幾題?

5 題選擇，每對 +4、每錯 −1（無不作答）。總分至少 12 分，至少對幾題?
設 對 x 題、錯 (5−x) 題　列式　4x − (5−x) ≥ 12
解　4x − 5 + x ≥ 12 → 5x ≥ 17 → x ≥ 3.4
取整數　最小整數解 x = 4
至少要答對 4 題
x ≥ 3.4，但題數是整數 → 最小整數 4。
例題
例題 8　難度 ★★☆　比較兩方案

通話幾分鐘以上選乙?

甲:月租 200、每分 0.4 元；乙:月租 350、每分 0.2 元。幾分鐘以上選乙較划算?
設 通話 x 分鐘，乙較划算 ⟺ 甲費用 > 乙費用
列式　200 + 0.4x > 350 + 0.2x
解　0.2x > 150 → x > 750
通話超過 750 分鐘時選乙較划算
「乙較划算」= 甲比較貴 = 甲 > 乙，列式方向要想清楚。
例題
例題 9　難度 ★★★　加權平均

期末考至少幾分?

平時 4 次共佔 40%（取平均）、期末佔 60%，加權要 ≥ 70。前 4 次 65、78、80、72，期末至少幾分?
平時平均　(65+78+80+72)/4 = 73.75
列式　73.75×0.4 + 0.6x ≥ 70 → 29.5 + 0.6x ≥ 70
解　0.6x ≥ 40.5 → x ≥ 67.5 → 取整數 68
期末考至少要考 68 分
例題
例題 10　難度 ★★★　情境上限

工廠生產與利潤

每天最多 200 條，每條利潤 80 元，固定成本 6000 元。(1) 損益兩平至少幾條? (2) 利潤超過 5000 元幾條?
(1) 80x − 6000 ≥ 0 → x ≥ 75 → 至少 75 條
(2) 80x − 6000 > 5000 → 80x > 11000 → x > 137.5 → 至少 138 條
檢查上限　138 < 200 ✓ 在每天上限內
⚠️ 別忘了檢查情境的隱藏上限（最多 200 條）。
核心
連立（雙邊）不等式

同時滿足兩條件 → 取交集

一個未知數要同時滿足兩個不等式 → 分別解出後取交集（AND）
⚠️ 常見錯誤:取聯集（OR）而非交集。
「同時成立」= 兩個範圍重疊的部分 = 交集。
x > 3
x ≤ 8
交集 = 3 < x ≤ 8
例題
例題 11　難度 ★★☆　連立

求整數解

x 的 2 倍加 3 大於 9，且 x 的 3 倍減 5 不超過 19，求整數解。
第一式　2x + 3 > 9 → x > 3
第二式　3x − 5 ≤ 19 → x ≤ 8
交集　3 < x ≤ 8
整數解:4、5、6、7、8（共 5 個）
x > 3 不含 3，x ≤ 8 含 8 → 整數從 4 到 8。
例題
例題 12　難度 ★★☆　雙邊不等式

解 −3 ≤ 2x − 1 ≤ 7

解 −3 ≤ 2x − 1 ≤ 7
三邊同 +1　−2 ≤ 2x ≤ 8
三邊同 ÷2（正數不翻）　−1 ≤ x ≤ 4
−1 ≤ x ≤ 4
雙邊不等式:三邊一起做同樣運算。除負數時三邊一起翻。
例題
例題 13　難度 ★★★　連續偶數

找三個連續偶數

連續三個偶數，最小的比最大的 2 倍少 22，求這三個偶數。
設 三個連續偶數 2n、2n+2、2n+4
條件　2n = 2(2n+4) − 22 → 2n = 4n − 14 → n = 7
14、16、18
驗算　14 = 2×18 − 22 = 36 − 22 = 14 ✓
素養
素養題型 1　【烤肉活動】

預算分配

場地費每人 50 元，雞翅 30 元/份、玉米 10 元/根，每人預算上限 180 元。食材預算 = 180 − 50 = 130 元
(1) 固定 2 根玉米，最多幾份雞翅?　30x + 20 ≤ 130 → x ≤ 3.67 → 最多 3 份
(2) 買 3 份雞翅，玉米最多幾根?　90 + 10x ≤ 130 → x ≤ 4 → 最多 4 根
(3) 各買 x 個，最多幾個?　40x ≤ 130 → x ≤ 3.25 → 各最多 3 個
素養
素養題型 2　【輪胎廠達標】

目標生產量

本月目標不少於 2000 條，已生產 1340 條，還有 x 天，每天最多 90 條。
(1) 還需至少幾條?　2000 − 1340 = 660 條
(2) 建立達標不等式（每天 y 條，y ≤ 90）　xy ≥ 660
(3) 還有 9 天，每天至少幾條?　9y ≥ 660 → y ≥ 73.3 → 至少 74 條
「不少於 2000」→ ≥ 2000（至少，含端點）。
避雷

避雷指南:六個最常見錯誤

❌ 錯誤✓ 正確
除以負係數忘記翻轉乘除負數，方向必翻（最重要！）
去分母只乘部分項每一項都乘 LCM（含常數）
「至少／不超過」翻反至少 = ≥（下限）、不超過 = ≤（上限）
解出小數直接當答案人數/件數取整數（最小/最大）
連立取聯集「同時成立」取交集
忽略情境隱藏上限檢查「最多 200 條」等限制
自我檢查:「這一步乘除的是正數還是負數?負數要翻轉！」
總結

本節總結

  • 解法流程:去括號 → 去分母 → 移項合併 → 除以係數 → 數線
  • ⚠️ 除以負係數那一步，不等號翻轉 —— 和方程式唯一的差別
  • 應用:圈臨界詞列式（超過/至少…）、人數件數取整數
  • 連立不等式:分別解，取交集（同時成立）
